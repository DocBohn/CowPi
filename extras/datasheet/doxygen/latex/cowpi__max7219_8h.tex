\doxysubsection{Detailed Description}
Utility functions for peripherals driven by MAX7219 such as 8-\/digit/7-\/segment display modules and 8x8 LED matrices. 

\begin{DoxyAuthor}{Author}
Christopher A. Bohn
\end{DoxyAuthor}
If {\ttfamily MAX7219} is part of the argument to {\ttfamily \mbox{\hyperlink{group__configuration_ga0195991b9a0da4e8d6db820b9cf87d2a}{cowpi\+\_\+setup()}}}, then the display module will be configured to use all eight digits (or all 8 rows in the case of an 8x8 LED matrix), intensity level 3 (7/32 of full brightness), no-\/decode, not shut-\/down, and not in test mode.

The MAX7219 is simple enough that firmware programs can send data (or commands) to it directly through {\ttfamily \mbox{\hyperlink{group__display_ga15e63ea649a3892b5e6e24ea17ae3fde}{cowpi\+\_\+max7219\+\_\+send()}}} (or its re-\/implementation).
