Identifies the \char`\"{}dialect,\char`\"{} or the mapping of protocol bits to display module bits, that the library has been configured to use. 

Some display modules ({\itshape e.\+g.}, MAX7219-\/based modules) have only one possible mapping. {\footnotesize\ttfamily bool cowpi\+\_\+is\+\_\+spi\+\_\+lsbfirst (\begin{DoxyParamCaption}\item[{void}]{ }\end{DoxyParamCaption})}



Identifies whether the SPI hardware will send the least-\/significant bit first. 

This function is unlikely to be called by the firmware program\+: there is no actionable reason for the program to do so. There is, however, no harm in doing so.

This function will return {\ttfamily true} only when the display module is {\ttfamily LCD1602} and the dialect is {\ttfamily ADAFRUIT}.

\begin{DoxyReturn}{Returns}
{\ttfamily true} if the display module and dialect require that the SPI protocol sends the least-\/significant bit first
