Reports whether the left switch is in the left position. 

There is no debouncing. This is a portable implementation, not a memory-\/mapped implementation.

Assumes the left switch is in Arduino pin A4 (D18) if SPI (but not I2C) is in use or if no protocol is in use; assumes the switch is in pin D11 if I2C (but not SPI) is in use. If both protocols are in use, then this function will always return {\ttfamily false}. A switch in the left position grounds a pulled-\/high input.

\begin{DoxyReturn}{Returns}
{\ttfamily true} if the switch is in the left position, {\ttfamily false} otherwise
