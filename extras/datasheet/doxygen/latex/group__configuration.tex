{\footnotesize\ttfamily void cowpi\+\_\+set\+\_\+display\+\_\+dialect (\begin{DoxyParamCaption}\item[{unsigned int}]{dialect }\end{DoxyParamCaption})}



Sets the \char`\"{}dialect,\char`\"{} or the mapping of protocol bits to display module bits. 

Some display modules ({\itshape e.\+g.}, MAX7219-\/based modules) have only one possible mapping, and calling this function has no effect for those modules. For other display modules, the {\ttfamily COWPI\+\_\+\+DEFAULT} dialect is the default; this function does not need to be called if the {\ttfamily COWPI\+\_\+\+DEFAULT} dialect will be used.

\begin{DoxyAttention}{Attention}
If this function is called, it must be called before {\ttfamily \mbox{\hyperlink{group__configuration_ga0195991b9a0da4e8d6db820b9cf87d2a}{cowpi\+\_\+setup()}}} so that the display module can be properly configured.
\end{DoxyAttention}

\doxysubsubsection{\texorpdfstring{cowpi\_set\_display\_i2c\_address()}{cowpi\_set\_display\_i2c\_address()}}
{\footnotesize\ttfamily void cowpi\+\_\+set\+\_\+display\+\_\+i2c\+\_\+address (\begin{DoxyParamCaption}\item[{uint8\+\_\+t}]{peripheral\+\_\+address }\end{DoxyParamCaption})}



Sets the I2C address for an I2\+C-\/driven display module. 

Because the I2C (aka IIC, TWI) protocol uses addresses to select the peripheral that the microcontroller is communicating with, the display module\textquotesingle{}s address needs to be set. If the SPI protocol is being used, then there is no need to call this function.

The common PCF8574-\/based interfaces used with LCD1602 display modules typically have an address of 0x27, but this can readily be changed.

\begin{DoxyAttention}{Attention}
If this function is called, it must be called before {\ttfamily \mbox{\hyperlink{group__configuration_ga0195991b9a0da4e8d6db820b9cf87d2a}{cowpi\+\_\+setup()}}} so that the display module can be properly configured.
\end{DoxyAttention}

Configures the microcontroller\textquotesingle{}s pins for the expected hardware setup, and configures the library to work with the specified display module and communication protocol. 

\begin{DoxyAttention}{Attention}
If a non-\/{\ttfamily COWPI\+\_\+\+DEFAULT} dialect is used, then {\ttfamily \mbox{\hyperlink{group__configuration_ga02e764c97862eca443df369048a62f25}{cowpi\+\_\+set\+\_\+display\+\_\+dialect()}}} must be called {\itshape before} {\ttfamily \mbox{\hyperlink{group__configuration_ga0195991b9a0da4e8d6db820b9cf87d2a}{cowpi\+\_\+setup()}}}.

If the {\ttfamily I2C} (or {\ttfamily IIC} or {\ttfamily TWI}) protocol is used, then {\ttfamily \mbox{\hyperlink{group__configuration_ga7d4f5d9ab81e3dd55134d880d4a99aa5}{cowpi\+\_\+set\+\_\+display\+\_\+i2c\+\_\+address()}}} must be called {\itshape before} {\ttfamily \mbox{\hyperlink{group__configuration_ga0195991b9a0da4e8d6db820b9cf87d2a}{cowpi\+\_\+setup()}}}.
\end{DoxyAttention}
If {\ttfamily printf()} and/or {\ttfamily scanf()} will be used, then we recommend that {\ttfamily \mbox{\hyperlink{group__configuration_ga8d3c4958d24333b91847fb26c8840542}{cowpi\+\_\+stdio\+\_\+setup()}}} be called before {\ttfamily \mbox{\hyperlink{group__configuration_ga0195991b9a0da4e8d6db820b9cf87d2a}{cowpi\+\_\+setup()}}}; however, this is not required.

{\footnotesize\ttfamily void cowpi\+\_\+stdio\+\_\+setup (\begin{DoxyParamCaption}\item[{unsigned long}]{baud }\end{DoxyParamCaption})}



Configures the Cow\+Pi library to use {\ttfamily stdio.\+h} functions. 

Configures {\ttfamily printf()} to write to, and {\ttfamily scanf()} to read from, the serial interface between the microcontroller and the host computer. Calls to {\ttfamily printf()} and {\ttfamily scanf()} that occur before the call to {\ttfamily \mbox{\hyperlink{group__configuration_ga8d3c4958d24333b91847fb26c8840542}{cowpi\+\_\+stdio\+\_\+setup()}}} will have no effect.
