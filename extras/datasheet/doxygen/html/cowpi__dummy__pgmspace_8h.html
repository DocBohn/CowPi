Detailed Description

This header defines dummy functions & macros to replace those in avr/pgmspace.h for non-AVR architectures (or that don't need avr/pgmspace.h, in the case of the ATMega4809).

Author
Christopher A. Bohn

This header might not be needed, as it looks like Arduino has provided a pgmspace.h header for some or all non-AVR architectures. When we extend the library to the Raspberry Pi Pico SDK, we may need it, though.
