Detailed Description

Utility functions for peripherals driven by MAX7219 such as 8-digit/7-segment display modules and 8x8 LED matrices.

Author
Christopher A. Bohn

If MAX7219 is part of the argument to cowpi_setup(), then the display module will be configured to use all eight digits (or all 8 rows in the case of an 8x8 LED matrix), intensity level 3 (7/32 of full brightness), no-decode, not shut-down, and not in test mode.

The MAX7219 is simple enough that firmware programs can send data (or commands) to it directly through cowpi_max7219_send() (or its re-implementation).
